双击打开PS
然后 双击插入图片
1.清除校准线  ctrl + H
2. Alt + 滑轮向上 放大图片
3. Alt + 滑轮向下 缩小图片
4.Ctrl + 滑轮向上  图片右移
5.Ctrl + 滑轮向下  图片左移

图片格式后缀名有  jpg  png  bmp  gif

psd ai 是UI设计给的图

Ctrl + R   调出标尺


截图：
选中图片后，如果大小不是很合适；右键--变换选区；
调节成合适的大小后，enter一下
ctrl + c 将选中区域复制
Ctrl+ N   enter 新建一个区域用来放图片
Ctrl+ V 将图片复制
ctrl+ s  保存

图片信息按F8显示   或者在 窗口 找  信息


Ctrl   N  新建文件
起名字 index.html

Ctrl S  保存
